Conan Exiles Database Transfers
=================================

Lightweight PySide6 utility to transfer items, buildings, thralls, and related assets between player characters in a Conan Exiles SQLite `game.db` file.

- Select a source and target character.
- Choose categories to transfer (items, buildings, thralls, game events).
- Optional backup before applying changes and an audit CSV is written for every transfer.
- A public domain database is included for testing purposes.
